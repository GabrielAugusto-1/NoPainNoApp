🚀 Criar vários produtos com JavaScript (no navegador)
Abre o site /produtos/novo no localhost:8080, aperta F12 → aba Console e cola:

for (let i = 1; i <= 10; i++) {
  // cria um form invisível
  let form = document.createElement("form");
  form.method = "POST";
  form.action = "/produtos/novo";

  // cria input nome
  let nome = document.createElement("input");
  nome.name = "nome";
  nome.value = "Produto " + i;
  form.appendChild(nome);

  // cria input valor
  let valor = document.createElement("input");
  valor.name = "valor";
  valor.value = (i * 10).toFixed(2);
  form.appendChild(valor);

  // adiciona ao body e envia
  document.body.appendChild(form);
  form.submit();
}

🔥 Criar + Excluir produtos automaticamente
async function criarEExcluirProdutos(quantidade) {
  // Criar produtos
  for (let i = 1; i <= quantidade; i++) {
    await fetch("/produtos/novo", {
      method: "POST",
      headers: {
        "Content-Type": "application/x-www-form-urlencoded"
      },
      body: `nome=Produto_${i}_${Date.now()}&valor=${(i*10).toFixed(2)}`
    });
    console.log("Criado: Produto_" + i);
    await new Promise(r => setTimeout(r, 200)); // espera 200ms
  }

  console.log("=== Todos os produtos criados ===");

  // Excluir produtos (assumindo IDs 1..quantidade)
  for (let i = 1; i <= quantidade; i++) {
    await fetch(`/produtos/excluir/${i}`, { method: "GET" });
    console.log("Excluído: Produto ID " + i);
    await new Promise(r => setTimeout(r, 200));
  }

  console.log("=== Todos os produtos excluídos ===");
}

// Executa com 5 produtos, por exemplo:
criarEExcluirProdutos(5);
✅ Como funciona
Cria quantidade de produtos com nomes únicos (Produto_X_timestamp)

Espera 200ms entre cada criação pra não travar o backend

Depois exclui os mesmos produtos pelo ID

Mostra logs no console

⚡ Dicas importantes

Pode aumentar quantidade pra criar 10, 20 ou 50 produtos
